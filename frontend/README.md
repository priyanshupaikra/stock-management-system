# Frontend Documentation

This README file contains all necessary information regarding the frontend component of the Stock Management System. It includes installation instructions, usage guidelines, and additional resources.

## Installation
To install the frontend dependencies, run:

```bash
npm install
```

## Usage
To start the frontend server, execute:

```bash
npm start
```
